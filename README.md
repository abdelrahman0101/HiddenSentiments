This is the code repository for the paper:

**Hidden Sentiments: The impact of low-level adversarial perturbations on Arabic sentiment analysis services**

<!-- Accepted for presentation at the OSACT7 workshop at LREC2026.-->

The files include the code used for generating the samples from the original corpora, the generated samples, the code used for generating perturbed versions, and the results of testing each service with each version.
